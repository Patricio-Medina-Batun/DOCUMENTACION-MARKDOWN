# Determinantes – Apuntes

## Ejercicio 1: Determinantes 2×2  
**Calcula los determinantes:**

### Matriz A

⎡ 5 2 ⎤
⎣ 3 1 ⎦

ΔA = 5·1 − 2·3 = **−1**

---

### Matriz B

⎡ -1 8 ⎤
⎣ 2 4 ⎦

ΔB = (−1)·4 − 8·2 = **0**

---

### Matriz C

⎡ 1 9 ⎤
⎣ 2 8 ⎦

ΔC = 1·8 − 9·2 = **0**

---

### Matriz D

⎡ 5 5 ⎤
⎣ 0 0 ⎦

ΔD = 5·0 − 5·0 = **0**

## Ejercicio 5: Aplicación geométrica

Dados los vectores  
u → = (3, 2)  
v → = (1, 4)

### a) Área del paralelogramo

⎡ 3 1 ⎤
⎣ 2 4 ⎦

Área = 3·4 − 1·2 = **10**

---

### b) Intercambiando los vectores

⎡ 1 4 ⎤
⎣ 3 2 ⎦

Área = 1·2 − 4·3 = **−10**

---
